# list-google-image
Show search image given google list

## Description
A simple web tool that takes a list of items with prices and generates Google Images search links for each item.

## Usage
1. Open `index.html` in your web browser
2. The page will load with sample data pre-filled
3. You can edit the list or paste your own data
4. Each line should be in the format: `* Item Name — Price`
5. Click "Generate Google Image Links" button
6. Click on "Search Images" for any item to open Google Images search in a new tab

## Input Format
Each line should follow this format:
```
* Item Name — Price
```

Example:
```
* Nannostomus Cenepa — RM123
* Gastrodremus C123 — RM56
* Corydoras Reticulatus — RM53
```

## Features
- Automatically parses item names and prices
- Generates Google Images search URLs for each item
- Clean, user-friendly interface
- Pre-loaded with sample fish data
- Opens search results in new tabs
